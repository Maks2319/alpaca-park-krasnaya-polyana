# Парк Альпак · Красная Поляна

Лендинг парка альпак на плато над Красной Поляной. React + Vite + framer-motion.

## Разработка

```
npm install
npm run dev
```

## Сборка и деплой на GitHub Pages

```
npm run build
npx gh-pages -d dist
```
